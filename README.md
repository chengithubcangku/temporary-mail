[English](README-en.md)

匿名的一次性邮箱，保护您的个人电子邮件地址免受垃圾邮件的骚扰。

[🧰 自建部署教程](deploy.md)

## 🎉 可以做什么？

- 用于注册不想暴露自己真实邮件地址的网站，以避免垃圾邮件的骚扰。
- 用于在同一个平台注册多个账号，而又不需要去注册多个邮箱。

## ⚠️ 注意

- **❗接收到的邮件内容仅能保留10天**
- **❗随机生成的邮箱地址任何人都可以使用，请勿用于注册重要账号**
